# -Agent-
搭建面向大模型与 Agent 场景的 Web 智能对话平台，集成流式输出、RAG 检索增强、智能体工具调用、语音交互功能，优化长会话交互与复杂问答效果。
# 多模态Agent智能对话平台
基于大模型 + RAG 检索增强的智能对话系统，支持流式输出、语音交互、Agent工具调用。

## 技术栈
- Vue3 + Vite
- SSE 流式通信
- Web Speech API
- ECharts / 组件懒加载 / 虚拟滚动

## 功能介绍
- 流式对话实时输出
- RAG 知识库检索与来源标注
- 智能体 Function Calling 可视化
- 语音输入与识别
- 长列表性能优化

## 项目说明
个人前端实战项目，负责整体页面开发、交互实现与性能优化。
